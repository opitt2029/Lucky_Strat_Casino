# Background assets

Put background images for the frontend theme in this folder.

Then edit `frontend/src/theme/backgroundTheme.js` and set the matching `image` value:

```js
app: {
  image: 'casino-main.jpg',
}
```

Recommended image sizes:

- Main app background: 1920x1080 or larger
- Login hero background: 1600x1200 or larger
- Mobile-friendly images should keep important content near the center
